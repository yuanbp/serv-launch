# 后台管理系统

## 简介:
简单的后台开发模板框架，具备菜单管理,用户管理,角色管理,权限管理,日志管理这5个功能。

运用layui后台管理模板，界面简介大方且美观。

采用mvc架构，使代码易于管理和修改，合理的注解，使代码通俗易懂，具有较强的阅读性。
